Toneelstuk over dictatorland

Dit is een toneelstuk dat ik geschreven heb over een land waar een dictator de macht heeft. In dat land is er een groep revolutionairen die macht probeert over te nemen. Ze zitten ondergedoken in een bar en bedenken een plan om de dictator te vermoorden. Maar later in het verhaal wordt het ook duidelijk dat de barman niet helemaal te vertrouwen is.

Ik had dit toneelstuk geschreven voor toneel. Helaas wilde de andere mensen die ook op toneel zaten het verhaal niet spelen. Als er iemand het toneelstuk wilt gebruiken in een toneelstuk mag hij/zij dat zeker doen. Het is niet verplicht, maar ik zou het leuk vinden als je dan contact met me opneemt, omdat ik het dan wel zou willen zien.

De rechtszaak van Sint-Nicolaas

Ik schreef dit verhaal in 2015. Ik zat toen in groep 8 en het was voor een schrijfopdracht voor school. Ik kan met trots zeggen dat ik deze schrijfopdracht gewonnen heb en ik mijn verhaal voor mocht lezen. Het verhaal stelt de vraag hoe Sinterklaas nou aan het geld komt om al die cadeautjes te kopen. Het blijkt dat Sinterklaas dit op illegale wijze doet. Hiervoor wordt Sinterklaas gearresteerd.

Mini Underground

Mini Underground is a remake of the popular game Mini Metro. You start with a few stations. It is your task to make the routes the trains must follow. You have to make sure that stations don't get overcrowded. The game has been designed for GNU/Linux and Android, but it probably won't be so hard to port it to Windows and MAC OS. The program is written in C++ using the SDL library.

Zobrollo

Zobrollo is a 2d minimalistic top-view racing game programmed in C using the Allegro library for Windows 10 and GNU/Linux. At this point of development you can only play in Time Trial. The game also keeps of track of records you have made and you are able to rewatch previous races and play against them.

AsciiMind

This is a command-line game based on the classic board game "Mastermind". It differs in that instead of colours it uses ASCII characters. This is nice, because there are more ASCII characters than distinguishable colours. What is also neat is that after every guess you make you see how many possible answers there are left.
